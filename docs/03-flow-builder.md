# Flow Builder

Flow Builder digunakan untuk menggambarkan Customer Journey.

Flow Builder BUKAN tempat membuat template.

Flow Builder BUKAN tempat membuat webhook.

Flow Builder hanya menghubungkan semuanya.

---

Contoh Flow

Start

-> Welcome

-> Waiting Poster

-> Poster Approved

-> Send Pricelist

-> Choose Package

-> Offer PIN

-> Payment

-> Payment Success

-> Ask Instagram

-> Queue

-> Done

---

Setiap Node memiliki:

- Nama
- Deskripsi
- Template
- Automation
- Next Step

Flow Builder hanya menyimpan urutan flow.

Tidak mengirim WhatsApp secara langsung.

Flow akan dipanggil oleh Automation.
