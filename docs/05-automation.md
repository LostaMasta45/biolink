# Automation

Automation menentukan kapan Template dijalankan.

Automation memiliki 3 bagian.

Trigger

-> Condition

-> Action

---

Contoh

Trigger

Customer First Message

-> Condition

Keyword = Halo

-> Action

Send Welcome Template

---

Action yang digunakan mengikuti kemampuan Kirim.dev.

Contoh:

- Send Template
- Send Quick Reply
- Delay
- Assign Label
- Update Status

Automation hanya menyimpan konfigurasi.

Eksekusi dilakukan oleh backend.
