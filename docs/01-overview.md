# Overview

Dashboard WhatsApp terdiri dari beberapa module.

```text
WhatsApp

|-- Overview
|-- Flow Builder
|-- Templates
|-- Automation
|-- Auto Reply
|-- Logs
|-- Webhook
`-- Settings
```

Dashboard hanya menyimpan konfigurasi.

Semua pengiriman pesan dilakukan menggunakan API Kirim.dev.

Webhook digunakan sebagai sinkronisasi data.
